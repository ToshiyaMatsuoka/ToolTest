
 **プログラムの実行前に**  

Test.sqlを実行してDB作成して下さい。  
`enemy`という名称のテーブルが作成され、
テストデータが書き込まれます。

---
このツールは敵キャラの  
・名前  
・ボスか否か  
・HP  
・MP  
・攻撃力  
・防御力  
・知力  
・精神力  
・レベル  
・経験値  
を設定し保存するデータベースです。

---
このアプリについてボタンで頑張ったところの内容を表示,  
LoadボタンでDBからデータ読み込み,  
SaveボタンでDBに表のデータを保存,  
行選択時にDeleteキーでDBと表からデータを物理削除します。
